01The problem

AI systems hallucinate — confidently stating things that aren't grounded, caving to misleading hints, or reasoning inconsistently.

The hard part

How do you detect hallucination when you don't have perfect ground truth — and how do you trust the detector itself?

A detector you haven't validated is just another model that might be wrong.

03The key insight

Validate the detector against known-good and known-bad synthetic targets before pointing it at any real model.

Known-good bot

Answers correctly, stays grounded, refuses when it should. Should not be flagged.

Known-bad bot

Hallucinates, invents claims, caves to hints. Designed to trigger the detector.

If the detector can't separate the bots we built, it can't be trusted on models we didn't.

04What's implemented today

RAG / groundedness live

  • Groundedness judge (lexical-overlap baseline) — primary
  • Independent novel-claim judge
  • → feeds the agreement diagnostic

Reasoning live

  • Answer correctness
  • Misleading-hint resistance
  • Explanation ↔ final-answer consistency
  • Overthinking heuristic

Runs deterministically today so the detector can be validated before connecting real models.

05The judge bank

Multi-signal by design — the primary judge drives the verdict, the rest are diagnostics.

  • Primary judge drives the flag decision.
  • Secondary judges contribute independent signals + a Judge Agreement metric.
  • Agreement is a diagnostic, not a quality score — judges can agree and still be wrong.
  • Crash containment: a judge that raises is caught. If the primary errors, the verdict is errored — never a silent pass.

Opt-in judges ready to wire in: NLI entailmentcitation faithfulnessself-consistency

07Reproducibility is built in

A report can't ship without its config.

Every RunSession export embeds:

  • Attacker / target / judge config
  • Expected-label mode
  • Golden source paths
  • Per-case seed + raw target output
  • Per-case suggested_fix on flagged/errored cases
  • Timestamp, git commit, Python, platform
  • Dependency versions

Exports as both Markdown (escaped) and JSON.

EXWhere the lexical baseline fails

On the synthetic oracle, FPR/FNR = 0 by design. On adversarial inputs the lexical judge — which counts word overlap, not entailment — breaks both ways.

False negative — missed

Answer parrots the context, then smuggles in an unsupported claim.

“…30-day clothing refund … and electronics get a full cash refund.”  → overlap 0.57 ≥ 0.45 → not flagged

False positive — false alarm

Correct answer, paraphrased with synonyms → few shared words.

ctx “15 vacation days” vs “three weeks of paid leave”  → overlap 0.00 < 0.45 → wrongly flagged

08Honest scope & limitations

CapabilityStatus
RAG groundedness — lexical-overlap baselinelive
Reasoning (correctness / hints / consistency)live
NLI entailment judge (needs transformers + torch)coded · deps off
Citation & self-consistency judgesopt-in
Image / VLM modalityinterface-ready · roadmap

Multi-hop and outdated-context prompts are included, but default detection still treats them as groundedness failures — not true multi-hop or staleness reasoning.

Baseline failure modes are documented as deliberately "wrong-on-purpose" regression tests — the targets to beat when NLI lands.

12Roadmap — swap synthetic → real

The interfaces don't change; only the implementations do.

  • NLI groundedness judge — replace lexical overlap with DeBERTa-MNLI entailment.
  • Real RAG target — Target.respond over Ollama + ChromaDB.
  • Adaptive attacker — an LLM that learns from past successes.
  • Image module (bonus) — image Target + VLM Judge; runner, report & dashboard unchanged.

Methodology: calibrate thresholds on synthetic bots first, then aim at real models.

13Takeaway

Mirage is a reproducible red-team & evaluation harness for hallucination and reasoning-failure detectors.

It earns trust the right way: validated against synthetic known-good / known-bad targets with real detector metrics before any production model is wired in.

This is a reproducible MVP harness — not a production hallucination detector yet.
